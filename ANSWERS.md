1. How to run

Requirements:
Node.js installed
npm installed

Steps:
git clone https://github.com/khubaib-ahmed419/pomodoro-app.git
cd pomodoro-app
npm install
npm run dev

2. Why I picked this stack

I used React because i am more familiar with this stack and it has modularity feature that allows to improve performance and maintainability.

I made the timer text very large (6rem) and centered it so users can read it easily.

3. Responsive and accessibility

On mobile:
Buttons wrap and timer stays readable

On desktop:
Timer stays centered and large

Accessibility:
High contrast UI and keyboard accessible buttons
